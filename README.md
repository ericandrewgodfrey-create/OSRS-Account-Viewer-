# OSRS-Account-Viewer

A 100% safe, open-source, RuneLite-powered website that lets players show off:
- Combat Achievements
- Achievement Diaries
- Treasure Trail counts
- Current Slayer task
- Equipment & last inventory

No bank data · No location · No real-time tracking · Fully user-controlled privacy

Built with:
- RuneLite plugin (Java)
- Node.js + Express + PostgreSQL backend
- React + Tailwind frontend (optional)
